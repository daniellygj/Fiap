# TDD

Exemplo simples de TDD

Será implementado um código simples tendo em vista as seguintes regras:

* Para pagamentos feitos à vista, em dinheiro, conceder 10% de desconto
* Para pagamentos feitos à vista, no cartão de débito, conceder um desconto de 5%
* Para pagamentos feitos no carsão de crédito (parcelado ou não), não conceder desconto
